机器学习类别：
## 按**学习方式**划分（最主流分类）

| 类别                                      | 核心思想                  | 典型算法                                                    | 是否适合本地部署                   | 典型应用（贴合你项目）                                     |
| --------------------------------------- | --------------------- | ------------------------------------------------------- | -------------------------- | ----------------------------------------------- |
| **1. 监督学习（Supervised Learning）**        | 从**带标签数据**中学习输入→输出的映射 | - 回归：线性回归、XGBoost<br>- 分类：SVM、随机森林、逻辑回归                 | ✅ 极适合（模型小、推理快）             | • 唤醒词检测（MFCC + SVM）<br>• 用户意图分类（“查天气” vs “写代码”） |
| **2. 无监督学习（Unsupervised Learning）**     | 从**无标签数据**中发现结构/模式    | - 聚类：K-Means、DBSCAN<br>- 降维：PCA、t-SNE<br>- 关联规则：Apriori | ✅ 适合（计算轻量）                 | • 用户语音行为聚类（高频问题分析）<br>• 声纹识别（无监督聚类）             |
| **3. 半监督学习（Semi-supervised Learning）**  | 少量标签 + 大量无标签数据        | - 自训练（Self-training）<br>- 图神经网络（GNN）+ 标签传播              | ⚠️ 中等（需一定算力）               | • 用少量标注语音 + 大量未标注录音提升 ASR 准确率                   |
| **4. 强化学习（Reinforcement Learning, RL）** | 智能体通过**试错 + 奖励**学习策略  | - Q-Learning、DQN<br>- PPO、SAC                           | ❌ 较难本地实现实时训练<br>✅ 可部署训练好策略 | • 对话策略优化（多轮交互奖励设计）<br>• 语音交互节奏控                 |
|                                         |                       |                                                         |                            |                                                 |
## 按**任务类型**划分（工程落地视角）

| 任务类型                             | 目标     | 典型算法/模型                        | 本地部署可行性                                   | 你的 AI 音响应用场景                       |
| -------------------------------- | ------ | ------------------------------ | ----------------------------------------- | ---------------------------------- |
| **分类（Classification）**           | 预测离散类别 | SVM、XGBoost、轻量 CNN             | ✅⭐⭐⭐⭐⭐                                    | • 唤醒词检测（二分类）<br>• 用户情绪识别（积极/中性/消极） |
| **回归（Regression）**               | 预测连续值  | 线性回归、LightGBM                  | ✅⭐⭐⭐⭐                                     | • 语音音量自适应调节<br>• 响应延迟预测            |
| **聚类（Clustering）**               | 发现数据分组 | K-Means、HDBSCAN                | ✅⭐⭐⭐⭐                                     | • 用户问题聚类 → 自动生成 FAQ                |
| **降维（Dimensionality Reduction）** | 压缩特征维度 | PCA、UMAP                       | ✅⭐⭐⭐⭐                                     | • 语音特征压缩 → 加速后续处理                  |
| **异常检测（Anomaly Detection）**      | 识别异常模式 | Isolation Forest、One-Class SVM | ✅⭐⭐⭐⭐                                     | • 检测恶意语音指令（如高频唤醒攻击）                |
| **序列建模（Sequence Modeling）**      | 处理时序数据 | HMM、LSTM、Transformer           | ⚠️ LSTM/Transformer 需 GPU<br>✅ HMM 可纯 CPU | • 语音端点检测（VAD）<br>• 多轮对话状态跟踪        |
