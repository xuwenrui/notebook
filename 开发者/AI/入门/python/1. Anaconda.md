#### 镜像配置
```
# 添加清华镜像源（主源 + free 源 + conda-forge）

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/

  

# 启用 show_channel_urls（排查问题用）

conda config --set show_channel_urls yes

  

# 禁用 SSL 验证（部分网络环境需要）

conda config --set ssl_verify false
```

```
# 1. 配置清华镜像（提速）

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

  

# 2. 创建你的 AI 环境

conda create -n ai-speaker python=3.11 -y

conda activate ai-speaker

  

# 3. 按需安装（仅 4 步）

conda install numpy pandas -y

conda install -c conda-forge onnxruntime-directml -y

pip install llama-cpp-python --config-settings=llama-cpp:gpu=directml

pip install pyaudio webrtcvad
```




| 操作           | 命令                             | 说明              |     |
| ------------ | ------------------------------ | --------------- | --- |
| **安装包**      | `conda install numpy pandas`   | 从 conda 源安装（推荐） |     |
| **用 pip 安装** | `pip install llama-cpp-python` | 当 conda 没有时用    |     |
| **更新包**      | `conda update numpy`           | 升级到最新兼容版        |     |
| **删除包**      | `conda remove numpy`           | 卸载              |     |
| **查看已装包**    | `conda list`                   | 当前环境所有包         |     |


| 操作         | 命令                  | 说明                   |
| ---------- | ------------------- | -------------------- |
| **清理缓存**   | `conda clean -a -y` | 删除下载的 tar 包（省 1~5GB） |
| **查看磁盘占用** | `conda info --envs` | 显示各环境路径 + 大小         |
|            |                     |                      |
