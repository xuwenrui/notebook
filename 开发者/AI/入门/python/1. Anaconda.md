#### 下载
- 官网：[https://www.anaconda.com/download](https://www.anaconda.com/download?spm=a2ty_o01.29997173.0.0.68ee51712swiTy)
建议安装Miniconda版本

| **Anaconda Distribution** | **Miniconda**                                                   |                                          |
| ------------------------- | --------------------------------------------------------------- | ---------------------------------------- |
| **本质**                    | **全家桶**：Python + Conda + 150+ 预装科学计算包（NumPy, Pandas, Jupyter 等） | **极简版**：仅 Python + Conda 包管理器，**无任何预装包** |
| **安装包大小**                 | ≈ **600 MB ~ 1.2 GB**（Windows）                                  | ≈ **60 MB ~ 80 MB**（Windows）             |
| **安装后磁盘占用**               | ≈ **3~5 GB**（含所有预装包）                                            | ≈ **300~500 MB**（仅基础环境）                  |
| **适合人群**                  | 新手、教学、快速开箱即用                                                    | 工程师、追求轻量、定制化环境                           |


```
conda list
conda env list

# 创建一个ai-develop的环境
conda create -n ai-develop python=3.11 -y
# 使用ai-develop环境
conda activate ai-develop


# 安装 Jupyter 
# 使用 `-c conda-forge` 是因为社区维护的包更新、更全，能避免很多依赖冲突。
conda install -c conda-forge jupyter -y
# 启动
jupyter notebook
```
#### 镜像配置


  






| 操作           | 命令                             | 说明              |     |
| ------------ | ------------------------------ | --------------- | --- |
| **安装包**      | `conda install numpy pandas`   | 从 conda 源安装（推荐） |     |
| **用 pip 安装** | `pip install llama-cpp-python` | 当 conda 没有时用    |     |
| **更新包**      | `conda update numpy`           | 升级到最新兼容版        |     |
| **删除包**      | `conda remove numpy`           | 卸载              |     |
| **查看已装包**    | `conda list`                   | 当前环境所有包         |     |


| 操作         | 命令                  | 说明                   |
| ---------- | ------------------- | -------------------- |
| **清理缓存**   | `conda clean -a -y` | 删除下载的 tar 包（省 1~5GB） |
| **查看磁盘占用** | `conda info --envs` | 显示各环境路径 + 大小         |
|            |                     |                      |
