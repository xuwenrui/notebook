### 安装
- 下载：https://gradle.org/install/
- 解压到指定目录并设置环境变量 `GRADLE_HOME` 指向解压后的目录。
- 配置 `PATH` 环境变量，包含 `%GRADLE_HOME%\bin`（Windows）或 `$GRADLE_HOME/bin`（Linux/Mac）。

### 常用命令
- **编译源代码**：    
    ```
    ./gradlew compileJava
    ```
    
- **运行测试**：
    ```
    ./gradlew test
    ```
    
- **打包项目**：
    ```
    ./gradlew jar
    ```
    
- **查看所有任务**：
    ```
    ./gradlew tasks
    ```
    
- **查看任务详情**：
    ```
    ./gradlew :taskName: --info
    ```
