[[模块化和maven的关系]]

##### 关键字
##### export
需要暴露的包，以供其他项目使用
``` java
// java.xml   module-info.class
module java.xml {  
    exports javax.xml;  
    exports javax.xml.catalog;
}
```
##### requires
```
// 调用方module-info.java中定义后才能使用
requires java.xml;
```
