###### pnpm
文档：[https://pnpm.io/zh/installation](https://pnpm.io/zh/installation)
```
# 获取当前配置的镜像地址
pnpm get registry
or
pnpm config get registry

# 设置新的镜像地址
pnpm set registry https://registry.npmmirror.com

npm install -g pnpm
```
