#### 配置
令行输入`nvm root`可以找到nvm安装路径
settings.txt中添加配置
```
node_mirror: https://npmmirror.com/mirrors/node/
npm_mirror: https://npmmirror.com/mirrors/npm/
```

#### 命令
```shell
# 查看网络可以安装的版本
nvm list available

# 查看已安装的node
nvm list
```
