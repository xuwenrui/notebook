安装npm install -g create-single-spa

#### 1. 创建 root-config（主项目）
npx create-single-spa
- 选择 root-config 类型
- 填写项目名（如 root-config）
- 选择包管理器（npm/yarn）
- 选择默认配置即可

#### 2. 创建 app-parcel（子应用）
npx create-single-spa
- 选择 single-spa application / parcel
- 选择 react（或你需要的框架）
- 填写项目名（如 app-parcel）
- 选择包管理器（npm/yarn）
