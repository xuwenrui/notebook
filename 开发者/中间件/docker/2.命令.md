```
{
  "builder": {
    "gc": {
      "defaultKeepStorage": "20GB",
      "enabled": true
    }
  },
  "experimental": false,
  "registry-mirrors": [
        "https://docker.anyhub.us.kg",
        "https://dockerhub.icu",
        "https://docker.awsl9527.cn"
  ]
}
```




###### 搜索远程镜像
docker search redis
###### 查看镜像版本
```
docker images

```


##### 容器
###### 启动镜像
```console
docker run -d -p 6379:6379 --name redis-server  redis:latest

-d 后台运行
-p 映射到本地ip
```

###### 查看容器
```bash
docker ps -a
docker ps -a|grep xxx
```

###### 删除容器
```
docker rm id/名称
或
docker rm -f id/名称
```
